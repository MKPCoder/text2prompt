# Prompt Builder (Streamlit)

A local web app to convert your input/context into a high-quality, structured prompt for:
- Create Dashboard
- Analyze Code
- Rewrite Code

## Quick start

1. (Optional) Create and activate a virtual environment.
2. Install dependencies:
   
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   
   ```bash
   streamlit run app.py
   ```

4. Open the local URL shown in the terminal (typically http://localhost:8501).

## Notes
- No external API keys required.
- Use the sidebar to choose task type, tone, and output format (plain text or JSON).
- Download the generated prompt as a .txt file.
